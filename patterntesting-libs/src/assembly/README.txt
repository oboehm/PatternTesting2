You'll find the following PatternTesting libraries in the lib directory:

    patterntesting-rt (PatternTesting Runtime)
    patterntesting-check-ct (PatternTesting Check-CompileTime)
    patterntesting-check-rt (PatternTesting Check-RunTime)
    patterntesting-concurrent (PatternTesting Concurrent)
    patterntesting-exception (PatternTesting Exception)

You'll find also other external libraries like commons-lang where the
PatternTesting libraries depends on.

The src directory contains the source jars which may be helpful for debugging.

Happy PatternTesting...
